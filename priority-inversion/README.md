Priority Inversion (No Protocol)
================================

Priority inversion demonstrations and debugging utilities without a protocol.

--------
- priority_inversion_taskset.py: Task set design to provoke inversion.
- priority_inversion_demo.png: Pre-generated inversion visualisation.
- debug_priority_inversion.py: Debug trace for inversion behavior.
- verify_resource_usage.py: Resource usage verification.
